# Cover Prompt Pack — Walking with Jesus

**Subtitle/positioning:** A 365-Day Devotional Journey with the Son
**Focus:** Jesus / discipleship, nearness, obedience, healing, following, abiding
**Palette:** midnight blue, parchment, olive green, dawn gold, clean white
**Symbol bank:** road toward sunrise, sandaled path, open Bible, shepherd imagery, table fellowship

## Shared Requirements
- Emotionally real; not generic AI-Christian art.
- Commercial devotional cover; readable at thumbnail size.
- Use elegant typography hierarchy: title first, subtitle second, author line: Susan “Lady D” Damon.
- Avoid plastic faces, fake smiles, cheap stock-photo worship poses, misspelled text, extra limbs/hands, and overdone glowing crosses unless concept specifically calls for a restrained cross.
- If generated with Leonardo.ai, use the highest available frontier-quality image model configured in the account; preferred model family if available: Nano Banana 2 / GPT Image 2 quality equivalent; portrait book cover aspect ratio.
